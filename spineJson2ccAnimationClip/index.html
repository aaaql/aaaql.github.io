請拖拉一個或多個 spine_xxx.json 至此 ...
spineJson 轉換 cc.AnimationClip 工具
說明：
- 新增 節點路徑:屬性 寫法 eg. node1:scale_120_node2:y_-10
- 目前支援的屬性 { fps, rotation, scale, scaleX, scaleY, opacity, x, y }
- time 0 要有初始值
- fps 預設為 24
- 有多個時 屬性與值 應成對 eg. scale_120_opacity_100_x_10_y_10
- 數值後處理
  - scale/100
  - scaleX/100
  - scaleY/100
  - opacity/100*255
